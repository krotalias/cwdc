codeplayer.js

Summary.

Codeplayer is a sandbox for writing HTML/CSS/JAVASCRIPT code.

Description.

There are three panels (textarea) to input the code and a fourth output panel (iframe) to display the result.

  • HTML panel
  • CSS panel.
  • Javascript panel.
  • Output panel.
