cgi.js

Simple output for CDC calculations.

Data supplied as URL parameters (query strings) and output into two <div> elements:
  • #rational (result and HTML Price table)
  • #result (discount and node Price table)

Usage:

  • np: número de parcelas
  • pv: preço à vista
  • pp: preço a prazo
  • pb: valor a voltar
  • nb: meses a voltar
  • e: entrada
  • prt: impressão
  • v: verbose
  • result.html?np=12&tax=4.55&pv=23000&pp=30354.50&pb=1000&e=0&prt=0&v=1

rational.cjs

Summary.

Desconto Racional por Dentro.

Coded following CommonJS module syntax, which was the original way to package JavaScript code for Node.js.

In fact, since this script is meant to be run on the browser (not on the server, via nodejs), it does not export anything. However, it does require two modules for using the nodejs CLI.
