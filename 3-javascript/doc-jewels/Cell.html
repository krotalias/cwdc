Class: Cell

Cell(row, col, icon)

Class that represents a grid position with an Icon. Optionally, it is possible to record a previous row for the icon.

Constructor

new Cell(row, col, icon)

Constructs a Cell with the given row, column, and Icon. The previous row is the same as the given row.
Parameters:

row Number row for this Cell.
