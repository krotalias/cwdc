factorize-ui.mjs

Summary.

Factorize UI main function for testing.

Calculates the Least Common Multiple up to the current MIN ≤ value ≤ MAX and displays its prime factors.

Output: 

                  [Z] ← → [X]  FIX: [SPACE]
|-------------------------------------O---------------------------------------------------------------| 37 = 53 bits
  LCM(37) = 2⁵ × 3³ × 5² × 7 × 11 × 13 × 17 × 19 × 23 × 29 × 31 × 37
A value the user requested: 37

Usage:
 - npm init
 - npm install readline-sync
 - node factorize-ui.mjs

To use ES6 modules in nodejs (version ≥ 13):

  1. file extension should be ".mjs"
  2. Add {"type": "module",} to package.json (located in current or parent directory
  3. should be used ES6 module syntax: import and export

gcd~gcd.js

Summary.

Package for calculating the Greatest Common Divisor and Least Common Multiple of whole numbers.

Description.

LCM returns the smallest whole number with all factors (divisible by all numbers) from 1 to a given limit.
