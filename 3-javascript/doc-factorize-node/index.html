Summary.

Factorize main function for testing.

argv:

  1. argv[0]: node path.
  2. argv[1]: script path.
  3. argv[2]: integer to be factorized.
Usage:
 - npm init
 - npm install readline-sync
 - node factorize.mjs 345 debug
