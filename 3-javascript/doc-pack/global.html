distance2Circles(p, l, inTest) → {Number|circle}

Return the distance between a point p given as a tuple (x,y), and the closest circle among all circles kept in the list l.

Each circle in l is represented as a tuple (x,y,r,c), where x,y are the coordinates of its center, r is the radius and c its color.

If p is inside a circle of radius r, this function returns the circle, if inTest is true, or 0 otherwise.

If l is empty, it returns a big number.
Parameters:
Name Type Default Description
p point a given point in the canvas.
l Array.<circle> circle list.
inTest Boolen false whether to search for the circle containing point p, or returning the distance from p to the closest circle.
