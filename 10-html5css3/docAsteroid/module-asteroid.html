(static) polar2Cartesian(r, a, dx, dy) → {Array.<Number, Number>}

Convert from polar coordinates to cartesian coordinates.

In fact, the polar system is rotated by -90°, and the angles are measured from the y axis (y → x, x → -y).

Note that:

  • sin(x) = -sin(-x) and cos(x) = cos(-x)
  • sin(90) = 1, sin(0) = 0, cos(90) = 0, cos(0) = 1
  • cos(A − B) = cos A cos B + sin A sin B
  • cos(A - 90) = sin(A)
  • sin(A − B) = sin A cos B − cos A sin B
  • sin(A - 90) = -cos(A)
  • R(90) = (-y, x)
  • R(-90) = (y, -x)
  • Equivalent to: polarToCartesian(r, a - Math.PI / 2, dx, dy)
Parameters:
Name Type Description
r Number polar distance.
a Number polar angle.
dx Number horizontal translation.
dy Number vertical translation.
