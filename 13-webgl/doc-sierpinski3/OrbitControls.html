Class: OrbitControls

OrbitControls(object, domElement)

Orbit controls allow the camera to orbit around a target.

To use this, as with all files in the /examples directory, you will have to include the file separately in your HTML.

Constructor

new OrbitControls(object, domElement)

OrbitControls constructor.
Parameters:
Name Type Description
object THREE.PerspectiveCamera | THREE.OrthographicCamera The camera to be controlled. The camera must not be a child of another object, unless that object is the scene itself.
domElement HTMLCanvasElement The HTML element used for event listeners.
