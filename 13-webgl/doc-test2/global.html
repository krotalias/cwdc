projection :Matrix4

For projection we can either use an orthographic projection (1), specifying the clipping volume explicitly, or a perspective projection (2), specified with a field of view, an aspect ratio, and distance to near and far clipping planes:

  • aspect ratio is 3/2 corresponding to a canvas size 600 x 400.
  • in (1) below, (right - left) = aspect × (top - bottom), corresponding to the screen aspect ratio.
  1. const projection = new Matrix4().setOrtho(-1.5, 1.5, -1, 1, 4, 6)
  2. const projection = new Matrix4().setPerspective(35, aspect, 0.4, 100)

(constant) viewMatrix :Matrix4

View matrix.

One strategy is to identify a transformation to our camera frame, then invert it.

Alternatively use the LookAt function, specifying the view (eye) point, a point at which to look, and a direction for "up".
Approximate view point for above is (1.77, 3.54, 3.06)
