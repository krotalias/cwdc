homework/hw3/test2.js

Summary.

Two cubes in a circular orbit. The smaller also orbits around the greatest as a satellite.

  • An animation of a colored unit cube orbiting counterclockwise is rendered about the y-axis, in the x-z plane, in a circle of radius 10.
  • The second cube, half the size of the first one, orbits clockwise around the first one in a radius of 2, and at three times the speed of the first one's orbit.
  • Neither cube should spin on its own axis.
  • The transformation is applied, passing the model matrix to the vertex shader.
Notes:
  1. the only modified code is in the draw function and in the vertex shader.
  2. it uses the type Matrix4 from the teal book utilities in cuon-matrix.js

lib/teal_book/cuon-matrix.js

Summary.

This is a class for handling 4x4 matrices.

It contains functions to create geometric transformations equivalent to OpenGL's:
  • glLoadIdentity
  • glRotate
  • glScale
  • glTranslate
  • gluLookAt
  • gluPerspective
  • glOrtho
  • glFrustum
This matrix is then right multiplied by one of those transformations.
