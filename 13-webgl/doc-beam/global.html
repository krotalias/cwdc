edgeRenderProgram(canvas) → {Object.<regl:Object, drawEdge:function>}

Return a regl object and a draw command with the source code for a vertex and a fragment shader.

A draw command wraps up all of the WebGL state associated with a draw call (either drawArrays or drawElements) and packages it into a single reusable function.
