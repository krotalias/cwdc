homework/hw2/Reflection.js

Summary.

Demo of matrix transformations.

  • An output area in the HTML page shows the matrices that were multiplied together to get the current transformation being applied to a triangle.
  • The drawing and shader code are the same as in Transformations.js
    What's been added are the HTML controls for selecting the transformation and the corresponding event handling code to update the transformation matrix.
  • Instead of explicitly listing 16 numbers to represent a matrix, we use the types Matrix4 and Vector4 from the teal book utilities to perform matrix operations.
  • The matrix stack has been removed from OpenGL version 3 and beyond.
  • Nonetheless, there are at least three packages available for matrix manipulation:
    1. glMatrix (available via CDN)
    2. cuon-matrix (examples/lib - unmaintained)
    3. MV (poorly documented)
    Of course, the choice is up to you...
  • This code is very old, and the use of var in Javascript has been "deprecated" meanwhile. Therefore, one should use const, and where not possible, let.
  • Do not spare objects in Javascript. They come for free and are very helpful, in general.
Usage example of a Matrix4: 
  var m = new Matrix4();           // identity matrix
  m.setTranslate(0.3, 0.0, 0.0);   // make it into a translation matrix
  var m2 = new Matrix4().          // create and make rotation in one step
    setRotate(90, 0, 0, 1);        // (rotate 90 degrees in xy-plane)
  m.multiply(m2);                  // multiply m on right by m2, i.e., m = m * m2;
  var theRealData = m.elements;    // get the underlying Float32Array
  var projection = new Matrix4().
    ortho(-1, 1, -1, 1, -1, 1);    // default projection

lib/teal_book/cuon-matrix.js

Summary.

This is a class for handling 4x4 matrices.

It contains functions to create geometric transformations equivalent to OpenGL's:
  • glLoadIdentity
  • glRotate
  • glScale
  • glTranslate
  • gluLookAt
  • gluPerspective
  • glOrtho
  • glFrustum
This matrix is then right multiplied by one of those transformations.
