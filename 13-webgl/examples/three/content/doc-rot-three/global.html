toNaturalFactor(f) → {Number}

Given a rational number f = n/d (float), returns the smallest integer k such that k * f becomes a natural number.

Write f in fraction form, then divide denominator by gcd(numerator, denominator).

  • E.g., for 30.25 = 3025/100, gcd is 25
  • Then return 100/25 = 4, that is,
  • 30.25 = 3025/25 / 100/25 = 121/4 ⇒ 30.25 * 4 = 121.

toNaturalFactor2(f) → {Number}

Dumb version of toNaturalFactor.

[2,4,5,6,8] and 10 have at least a common multiple with x < 10
  • (4 * x) % 10 == 0 ⇒ (2 * x) % 5 == 0 ⇒ x = 5 < 10
  • (4 * x) ☰ 0 (mod 10) ⇒ (2 * x) ☰ 0 (mod 5) ⇒ x = 5 < 10
  • 2 → [5], 4 → [5], 5 → [2,4], 6 → [5], 8 → [5]
[1,3,7,9] and 10 have all common multiples with x ≥ 10
  • (3 * x) % 10 == 0 or (7 * x) % 10 == 0 ⇒ x ≥ 10
  • (3 * x) ☰ 0 (mod 10) or (7 * x) ☰ 0 (mod 10) ⇒ x ≥ 10
Threfore, we can use the last digit of the float number to determine the smallest integer multiplier that turns it into a natural number. Of course the same rationale holds for 100, 1000, etc., and we only need to search for smaller factors when the number ends in 2,4,6,8. The other cases will always have a the smallest factor equal to 10, 100, 1000, etc.

E.g:

  • 3.25 * f = i ∈ ℕ
  • 3.25 * f = i ∈ ℕ
  • 325/100 * f = i
  • 325 = i * 100/f
  • 100/f ∈ ℕ ⇒ f is a factor of 100
  • 3.25 * 4 = 13
  • 325 * 0.04 = 13
  • 325 / 13 = 25 = gcd(325,100)
