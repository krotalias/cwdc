homework/hw3/test.js

Summary.

Similar to the two cubes example.

There are two cubes and an X. The cubes also orbit around the X as satellites.

  • An animation of a colored X orbiting clockwise is rendered about the y-axis, in the x-z plane, in a circle of radius 10.
  • The first cube is half of a unit cube and orbits clockwise around X with a radius of 8, while the second cube is one third of a unit cube and orbits counter-clockwise around X with a radius of 6.
  • Both cubes orbit at a constant angular speed of π radians/s or 180°/s.
  • The X also spins on its own y-axis "rev" times per full orbit, where "rev" is given in the URL (default 365).
  • The transformation is applied passing the model matrix to the vertex shader.
Notes:
  1. the only modified code is in the draw and animate functions, plus the vertex shader.
  2. it uses the type Matrix4 from the teal book utilities in cuon-matrix.js

lib/teal_book/cuon-matrix.js

Summary.

This is a class for handling 4x4 matrices.

It contains functions to create geometric transformations equivalent to OpenGL's:
  • glLoadIdentity
  • glRotate
  • glScale
  • glTranslate
  • gluLookAt
  • gluPerspective
  • glOrtho
  • glFrustum
This matrix is then right multiplied by one of those transformations.
