TSL Tutorial Part 1

This is a simple WebGPU application that uses Three.js and the Three.js Shading Language (TSL) to create a textured cube that rotates in 3D space. The application uses the WebGPURenderer to render the scene and the OrbitControls to allow the user to interact with the camera. The application also includes a GUI to set its parameters.
