This program provides code templates for use by Open\+GL developers. Although simple, the code supplies everything a beginner in Computer Graphics should know\+:


\begin{DoxyItemize}
\item Basic \href{../../OpenGL/surfaces.pdf}{\texttt{ math}}
\end{DoxyItemize}
\begin{DoxyEnumerate}
\item Surface \href{http://www.hao-li.com/cs599-ss2015/slides/Lecture02.1.pdf}{\texttt{ Implicit representation}}
\item Surface \href{http://tutorial.math.lamar.edu/Classes/CalcIII/ParametricSurfaces.aspx}{\texttt{ Parametric representation}}
\item \href{https://www.khanacademy.org/math/linear-algebra/vectors-and-spaces/dot-cross-products/v/normal-vector-from-plane-equation}{\texttt{ Normal vector}} calculation
\item \href{https://en.wikipedia.org/wiki/Quadric}{\texttt{ Quadrics}}
\item \href{https://blogs.scientificamerican.com/roots-of-unity/a-few-of-my-favorite-spaces-the-torus/}{\texttt{ Torus}} (ring, horn or spindle)
\end{DoxyEnumerate}
\begin{DoxyItemize}
\item Several render styles\+:
\end{DoxyItemize}
\begin{DoxyEnumerate}
\item \href{https://knowledge.autodesk.com/support/autocad/getting-started/caas/CloudHelp/cloudhelp/2016/ENU/AutoCAD-Core/files/GUID-84E193D7-A18D-4EE2-B978-19E4AFBCAEEC-htm.html}{\texttt{ Wireframe}}
\item \href{http://graphics.wikia.com/wiki/Gouraud_shading}{\texttt{ Smooth shading}}
\item \href{https://www.textures.com}{\texttt{ Texture}} (procedural or image based)
\end{DoxyEnumerate}
\begin{DoxyItemize}
\item \href{http://mathworld.wolfram.com/NormalVector.html}{\texttt{ Surface normal}} rendering
\end{DoxyItemize}
\begin{DoxyEnumerate}
\item drawn proportionally to the bounding box diagonal
\item pointing outside the object
\end{DoxyEnumerate}
\begin{DoxyItemize}
\item \href{https://en.wikipedia.org/wiki/Minimum_bounding_box}{\texttt{ Bounding Box}} computation
\item \href{https://msdn.microsoft.com/en-us/library/hh920765(v=vs.85).aspx}{\texttt{ Timer}} for controlling basic animation
\end{DoxyItemize}
