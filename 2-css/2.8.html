This text is in a <div> pointed by:

                #first-section {
                     background-color: #CEF6CE;
                     width: 260px;
                     bottom: 0;
                     right: 0;
                     zindex: 1;
                     /* (does not scroll) */
                     position: fixed;
                }

The quick brown fox jumped over the lazy dog.

These texts are in another <div> pointed by: 

                #second-section {
                     background-color: #8AB4F7;
                     width: 400px;
                     height: 5000px;
                     padding: 0;
                     top: 70px;
                     left: 20px;
                     /* (offset - top.., relative to itself ) */
                     position: relative;
                }

and these words are underlined.

Because the first-section has a z-index of +1, it will be placed above the second-section in a mobile device.
