Responsive Web Design

  • Bootstrap’s grid system uses a series of containers, rows, and columns to layout and align content.
  • It’s built with flexbox and is fully responsive.
  • Its grid system allows up to 12 columns across the page, which can be grouped together to create wider columns.
  • Grid columns should add up to twelve for a row. More than that, columns will stack no matter the viewport.

Usage

Column classes indicate the number of columns you’d like to use out of the possible 12 per row. Therefore, for three equal-width columns across, just use .col-4, or .col-md-4 to stack columns at md (768 px) breakpoint.
